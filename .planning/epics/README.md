# Epics

Epics group related atomic plans for a macro-phase defined in `ROADMAP.md`.
They do not replace `ROADMAP.md` or `.planning/plans/`; they sit between them.

Each epic is a markdown file describing:
- the goal of the epic
- the list of atomic plans it contains (by reference, e.g., `../plans/PLAN-XXX-*.md`)
- dependencies between plans
- acceptance criteria for the epic as a whole
